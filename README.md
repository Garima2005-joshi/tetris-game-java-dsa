# 🎮 Tetris Game Using Java Swing and DSA

A desktop implementation of the classic **Tetris Game** developed using **Java Swing** and **Data Structures & Algorithms** concepts such as **Stack** and **Queue**.

---

## 📌 Project Overview

This project demonstrates:

- Java Swing GUI Development
- Event-Driven Programming
- Stack implementation for Undo functionality
- Queue implementation for Next Piece Preview
- Collision Detection
- Game Loop using Timer

---

## 🛠️ Technologies Used

- Java
- Java Swing
- Eclipse IDE
- Stack
- Queue

---

## ✨ Features

- Seven Tetromino Shapes
- Piece Rotation
- Line Clearing
- Score System
- Pause and Resume
- Restart Game
- Undo Functionality
- Next Piece Preview
- Game Over Detection

---

## 🎮 Controls

| Key | Action |
|-----|---------|
| ← | Move Left |
| → | Move Right |
| ↑ | Rotate Piece |
| ↓ | Move Down Faster |
| P | Pause / Resume |
| U | Undo |

### Queue
Used for managing the **Next Piece Preview**.
