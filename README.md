TETRIS GAME USING JAVA SWING AND DSA



Project Overview



This project is a desktop implementation of the classic Tetris game developed using Java Swing. The game demonstrates the practical implementation of Data Structures and Algorithms concepts such as Stack and Queue.



Technologies Used



\* Java

\* Java Swing

\* Eclipse IDE



Data Structures Used



Stack



Used for implementing the Undo functionality. Every time a piece is locked on the board, the current board state is pushed into a stack. Pressing Undo restores the previous game state.



Queue



Used for managing the Next Tetromino Preview. The next piece is stored in a queue and displayed to the user before it appears on the board.



Features



\* Seven Tetromino Shapes

\* Piece Rotation

\* Line Clearing

\* Score System

\* Pause and Resume

\* Restart Game

\* Undo Functionality

\* Next Piece Preview

\* Game Over Detection



Controls



\* Left Arrow → Move Left

\* Right Arrow → Move Right

\* Up Arrow → Rotate Piece

\* Down Arrow → Move Down Faster

\* P → Pause/Resume

\* U → Undo
